# VibeSync Handover Screenshots

Put only redacted, repo-safe handover screenshots in this folder.

Do not commit screenshots that reveal:

- full API keys, secrets, service role keys, private keys, or recovery codes
- full credit card numbers, CVV, bank account numbers, or tax IDs
- passwords, password manager item contents, or 2FA QR codes
- private certificates, `.p8` files, `.env` values, or signing materials

Sensitive screenshots should live in the private handover packet or password manager, not in this repository.
