# VibeSync AI 鍵盤快速回覆規格

> 2026-07-16 Eric 定稿。第一版是「單則訊息驅動的 AI 回覆鍵盤」，不是完整 analyze-chat，也不是鍵盤版 Coach 1:1。

## 使用者結果

使用者第一次進入功能時，VibeSync 以動畫／逐步教學引導他前往 iPhone 設定，開啟 VibeSync 鍵盤與「允許完整取用」，回 App 後再教他長按地球切換鍵盤。

在 LINE、Instagram 等聊天 App 中，使用者：

1. 複製對方的一則訊息。
2. 切換至 VibeSync 鍵盤，主動點「載入訊息」。
3. 選擇五種風格之一：延展、共鳴、調情、幽默、冷讀。
4. 約 1–2 秒取得一則正體中文短回覆。
5. 回覆插入聊天 App 輸入框；VibeSync 絕不自動送出。

## V1 邊界

- Context 只來自使用者主動載入的剪貼簿文字；不讀取整段聊天、螢幕內容或鍵盤輸入歷史。
- 單次輸入 1–2000 字，單次輸出一則、最多 100 字。
- 五風格沿用 analyze-chat 的使用者語意，但走獨立 `keyboard-reply` Edge Function，不 import 或修改 analyze-chat。
- 生成成功固定共用 1 次月／日配額；未登入、非法輸入、quota／rate limit、模型或格式失敗均不扣額度。
- 生成內容只插入 `textDocumentProxy`，由使用者自行按聊天 App 的送出鍵。
- 未允許完整取用時，不呼叫網路或剪貼簿；保留基本文字輸入、退格與切換下一個鍵盤。
- 鍵盤不放購買流程。額度用完只顯示中性狀態並請使用者回 VibeSync 查看方案。

## 隱私與信任文案

在跳往 iPhone 設定前必須先說明：

> VibeSync 只會將你主動點擊「載入」的文字送去產生回覆，不會自動讀取或蒐集其他輸入內容。

登出、刪除帳號或 session 過期後，鍵盤不得繼續使用舊 token。鍵盤端不保存 refresh token，也不自行刷新 Supabase session。

## 體感與驗收

- 1–2 秒是正常網路下的 p50 體感目標；載入與生成必須有立即回饋，逾時可重試但不可重複扣費。
- 首次教學包含：前往設定、開啟鍵盤、允許完整取用、長按地球切換、複製→載入→選風格→插入。
- 設定頁保留「VibeSync AI 鍵盤」入口，可重看教學。
- 真機驗收涵蓋五風格、換一則、401、quota 429、model-rate 429、無網路、未允許完整取用、登出與 token 過期。
