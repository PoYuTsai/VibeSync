# Phase C - Growth Content Engine

> 狀態：規劃中
> 最後更新：2026-04-03

## 這個 Phase 在做什麼

這個 Phase 是上線後的內容與流量成長引擎。

核心方向：

- 經營 IG / Threads 品牌帳號
- 建立內容排程、自動產圖文、內容主題池
- 逐步把內容行銷從手動流程，變成半自動或自動化工作流

## 初步目標

- 建立品牌帳號內容系統
- 能固定排程發文
- 能產出符合品牌語氣的圖文素材
- 能追主題、tag、熱門議題
- 能找到潛在客戶互動點

## 目前想到的子題

### 內容生產

- 發文主題池
- 內容月曆 / 排程
- 自動產文案
- 自動產圖片 / 視覺素材
- 品牌語氣一致性

### 流量與互動

- 海巡必要 tag / 關鍵字
- 找潛在客戶
- 下方留言互動
- 發文 / 留言當作廣告型流量入口

## 之後要先討論清楚的規格問題

- 內容要全自動，還是人審後再發？
- 自動留言的風險邊界在哪？
- 品牌聲音要偏專業、曖昧教練、還是親切陪伴？
- 圖文生成是走模板化，還是完全生成？
- 要不要先只做「內容排程 + 草稿生成」，不直接做自動互動？

## 非本 Phase 重點

- 不處理 App 核心 OCR / 訂閱 bug
- 不處理 LINE OA 客服自動化
- 不處理 Android / Google Play 擴張

## 下一步建議

- 先和夥伴討論品牌目標與內容策略
- 再開一份正式 spec：
  - 目標受眾
  - 品牌語氣
  - 發文頻率
  - 素材來源
  - 自動化程度
- 然後再拆 implementation plan
