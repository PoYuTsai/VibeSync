# 定價方案 V2：訊息制 (最終版)

> **決定日期：2026-02-26**
> **策略：合理定價，不用之後漲價**

## 為什麼改用訊息制？

| 點數制問題 | 訊息制解決 |
|-----------|-----------|
| 長對話用戶覺得虧 | 用多少付多少 |
| 我們成本不可控 | 成本與收入正比 |
| 用戶難以預估 | 看得到剩餘額度 |

---

## 最終定價方案

### 核心邏輯
```
1 則訊息 = 1 訊息額度
每次分析消耗的額度 = 貼上的訊息數量
```

### 定價方案 ✅ 最終版
