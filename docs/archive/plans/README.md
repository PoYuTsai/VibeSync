# Archived Plans

This folder preserves shipped, superseded, or historical planning docs.

Do not use this folder as the default entrypoint for new work. New sessions should start with:

- `docs/plans/README.md`
- `docs/snapshot.md`
- `docs/decisions.md`
- `docs/shared-agent-rules.md`

If an archived plan becomes active again, move it back to `docs/plans/` and update `docs/plans/README.md`.
