## 2. 技巧名詞層（詞彙表＋顯現規則＋Apple 三層線）

**詞彙表**（prompt 新節，放 12 場景判斷區旁）：18 處標籤＋既有概念蒸餾成 **8–10 個**，每個一行：`名稱 → 一句定義 → 何時用 → 一個反例`。候選：價值展示、模糊邀約、合作框架、約會幻想、吐槽冷讀、失格（自嘲式降壓）、推（冷處理拉升）、不自證、框架維持。

**顯現規則（硬指令）**：analysis／avoid／rationale 用到技巧時**必須標技巧名＋一句為什麼**；few-shot 兩範例的 analysis 同步改寫成帶標注版（從示範學行為）。**B 砍刀教訓反向操作：名詞有顯現出口才允許進 prompt**（零顯現散文已實證是死重）。

**Apple 三層線（prompt 禁令區）**：
1. 輸出可見白名單：上述中性中文詞，全部安全
2. 內部 only：紅藥丸、PUA、IOI/IOD、shit test（輸出一律轉寫「試探」）——可引導判斷，絕不出現在輸出
3. 絕不出現（含 prompt）：性暗示技巧、物化/歧視語彙——也是未來餵 game 資料的過濾標準

技巧名顯現位置＝**既有欄位內嵌文字**（純 server-side，現有 TF build 立刻生效、零 client 工）；結構化技巧欄位（chip UI）留體系驗證後另案。
