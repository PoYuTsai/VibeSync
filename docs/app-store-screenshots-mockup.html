Design Spec

交付給設計時的規範摘要。

配色 Palette

  • 底：近黑 #0E0F13
  • 熱度漸層：#FF7A59 → #FF4D8D
  • 冷向（延展/主動性）：#4DA6FF
  • 警示：琥珀 #FFB43C 系

字體與版式 Type

  • 標語：繁中粗黑體 Bold（Noto Sans TC / 思源黑體）
  • 單張標語 ≤ 12 字、不超過兩行
  • 標語佔上 1/3，UI 截圖佔下 2/3
  • 數字用等寬（tabular-nums）

紅線 Guardrails

  • 禁：把妹／約砲／操控／套路／保證結果／代傳訊息
  • 不宣稱 OCR 準確率數字
  • 隱私不用絕對句
  • 價格以 pricing-final.md 為準
