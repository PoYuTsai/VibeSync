# /round

Create a short handoff for the current work round.

Rules:

- Use only the current conversation and already known facts.
- Do not read files or run commands unless the user asks.
- Do not include long logs, diffs, or pasted context.
- Keep it short enough to paste into a fresh session.

Format:

```text
Task:
Current state:
Changed:
Tests:
Risks:
Next:
Context to load:
```
